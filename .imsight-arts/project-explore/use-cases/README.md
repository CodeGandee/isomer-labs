# Isomer Labs Use Cases

This directory contains scenario use cases for semi-automatic research with Isomer Labs. Each use case applies the current platform language: Research Topic, Measurable Objective, Research Inquiry, Research Inquiry Relationship, Research Task, Parallel Execution Scope, Topic Workspace, Operator Agent, Project Operator Session, Domain Agent Team Template, Topic Team Specialization, Topic Team Instantiation Packet, Topic Agent Team Profile, Topic Agent Team Profile Bundle, Agent Team Instance, Task Handler, Run, Control Mode, Manual Mode, Completion Watcher Contract, Signal Observation, Artifact, Evidence Item, Decision Record, Gate, View Manifest, GUI Backend, GUI Layout Spec, AG-UI Render Payload, and AG-UI Event Envelope. Each use case includes a Mermaid use case diagram and a system-perspective Mermaid sequence diagram.
