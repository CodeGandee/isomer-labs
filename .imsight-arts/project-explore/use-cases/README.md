# Isomer Labs Use Cases

This directory contains scenario use cases for semi-automatic research with Isomer Labs. Each use case applies the current platform language: Research Thread, Research Goal, Research Task, Isomer Workspace, Operator Agent, Agent Team Template, Agent Team Instance, Task Handler, Run, Artifact, Evidence Item, Decision Record, Gate, View Manifest, GUI Backend, GUI Layout Spec, AG-UI Render Payload, and AG-UI Event Envelope. Each use case includes a Mermaid use case diagram and a system-perspective Mermaid sequence diagram.
