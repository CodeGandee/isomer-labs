# Docs

## Purpose

Human support views for the DeepResearch loop. Orientation only — not source authority.
`../manifest.toml` is the authoritative artifact index and `../specs/collab/collab-overview.md` is the
process authority.

## Contents

- `start-runbook.md`: exact ordered, tagged (read-only / state-mutating / launch-starting) commands to take the package live, with rollback/teardown. Not executed until operator-approved.
- `credentials.md`: credential / OAuth posture for the Claude-backed agents.
- `claude-effort.md`: Claude reasoning-level (effort) mapping; applied as a launch-time `--reasoning-level` override.
- `research-contract.md`: pre-launch research-contract expansion rubric (the operator-approved done-bar gate).
- `publication-quality.md`: scholarship bar + publication-grade output expectations and the packs that back them.
