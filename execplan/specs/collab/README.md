# Collaboration Specs

## Purpose

Process-first authority for the DeepResearch loop and its machine-readable topology. Every other
generated contract and artifact is derived from this directory.

## Contents

- `collab-overview.md`: the canonical process model (roles, stages, handoffs, continuation, terminal/recovery posture).
- `topology/topology.toml`: the machine-readable tree-loop topology (mode, root, routes, continuation edge, tool edges).
