# Show DeepSci Command Map

## Workflow

1. Load this exhaustive map only for `show-command-map`, `help`, or explicit complete output.
2. Present every current `isomer-ext-deepsci-entrypoint` public command exactly once in manifest order.
3. Preserve the user's task context in the exact example and state that invoking a pass begins governed research work.
4. Recommend a task-only entrypoint request when no named pass fits better.

If the goal does not map cleanly to one command, use the native planning tool to compare the pass boundaries and recommend a task-only DeepSci entrypoint request or ask for the missing output goal.
