# Research Paradigm Skills

This subtree contains portable research-stage skills for Isomer Labs agents.
The skills preserve research methods from the source analysis under
`context/explore/deepscientist-skill-analysis/` while removing source runtime
APIs, concrete storage layouts, scheduler behavior, and team-topology details.
