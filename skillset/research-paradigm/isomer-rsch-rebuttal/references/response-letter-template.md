# Response Letter Template

Use this template for rebuttal-ready author responses. Adapt it to the actual venue and response format.

## Drafting Rules

Write in a calm, direct, precise author voice. Brief professional courtesy is allowed, but move to substance immediately. Do not default to conceding fault. Use selective concede, selective clarify, and selective defend. Answer the reviewer concern directly in the first one or two sentences. Keep each response body to one or two focused paragraphs unless the response format requires otherwise.

Do not use bullets, label-value schemas, or private strategy language inside the actual response paragraphs. Do not rely on future edits alone when you can already give the clarification, evidence, argument, or replacement wording. If a concrete number, setup detail, or result is still unknown, use `[[AUTHOR TO FILL]]` rather than inventing specifics.
