# Candidate Ranking

Use this reference to choose a route by criteria rather than enthusiasm. Placeholder definitions live in `../migrate/placeholders.md`.

## Workflow

When this reference is used, execute the following steps in order.

1. **Rank by expected gain, mechanism plausibility, cost, implementation surface, comparator fit, novelty, and falsifiability**.
2. **Name the winner and the rejected alternatives**.

If the user's task does not map cleanly to these steps, use your native planning tool to build a step-by-step plan from this reference, the parent skill, and the available evidence, then execute the plan.
