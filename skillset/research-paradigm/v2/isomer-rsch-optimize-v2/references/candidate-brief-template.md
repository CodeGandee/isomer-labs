# Candidate Brief Template

Use this reference to shape loose methods into comparable briefs. Placeholder definitions live in `../migrate/placeholders.md`.

## Workflow

When this reference is used, execute the following steps in order.

1. **Include bottleneck, mechanism, family, change layer, expected gain, implementation surface, risks, foundation, and next target**.

If the user's task does not map cleanly to these steps, use your native planning tool to build a step-by-step plan from this reference, the parent skill, and the available evidence, then execute the plan.
