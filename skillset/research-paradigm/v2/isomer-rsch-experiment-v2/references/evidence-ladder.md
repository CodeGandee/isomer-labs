# Evidence Ladder

Use this reference to choose a target proportional to the claim. Placeholder definitions live in `../migrate/placeholders.md`.

## Workflow

When this reference is used, execute the following steps in order.

1. **Minimum means executable and comparable**.
2. **Solid means strong enough to carry the main claim**.
3. **Maximum means polish after credibility exists**.
4. **Do not spend for maximum before solid evidence exists**.

If the user's task does not map cleanly to these steps, use your native planning tool to build a step-by-step plan from this reference, the parent skill, and the available evidence, then execute the plan.

## Guidance

- Minimum means executable and comparable.
- Solid means strong enough to carry the main claim.
- Maximum means polish after credibility exists.
- Do not spend for maximum before solid evidence exists.
