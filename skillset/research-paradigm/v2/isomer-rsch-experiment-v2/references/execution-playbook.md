# Execution Playbook

Use this reference to execute one bounded run while preserving evidence. Placeholder definitions live in `../migrate/placeholders.md`.

## Workflow

When this reference is used, execute the following steps in order.

1. **Use planning surfaces when work is non-trivial**.
2. **Keep the comparator read-only**.
3. **Use smoke checks only when they reduce run risk**.
4. **Preserve command ids, logs, configs, outputs, seeds, and environment facts**.

If the user's task does not map cleanly to these steps, use your native planning tool to build a step-by-step plan from this reference, the parent skill, and the available evidence, then execute the plan.
