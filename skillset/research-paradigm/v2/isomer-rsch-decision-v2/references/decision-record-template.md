# Decision Record Template

Use this reference to record a consequential route decision durably. Placeholder definitions live in `../migrate/placeholders.md`.

## Workflow

When this reference is used, execute the following steps in order.

1. **Include route question, verdict, action, evidence basis, rejected alternatives, decisive reason, blocker if any, and next route**.
2. **Do not repeat the same decision without new evidence**.

If the user's task does not map cleanly to these steps, use your native planning tool to build a step-by-step plan from this reference, the parent skill, and the available evidence, then execute the plan.
