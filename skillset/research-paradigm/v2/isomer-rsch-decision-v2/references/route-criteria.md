# Route Criteria

Use this reference to judge routes from evidence rather than optimism. Placeholder definitions live in `../migrate/placeholders.md`.

## Workflow

When this reference is used, execute the following steps in order.

1. **Name support, contradiction, cost, risk, reversibility, and user-preference sensitivity**.
2. **For paper routes, apply publishability and exploration-depth gates before write or finalize**.
3. **For algorithm-first routes, consult the optimization frontier before overriding it**.

If the user's task does not map cleanly to these steps, use your native planning tool to build a step-by-step plan from this reference, the parent skill, and the available evidence, then execute the plan.
