# Evidence Recording

Use this reference to record scientific evidence as typed nodes before claims. Placeholder definitions live in `../migrate/placeholders.md`.

## Workflow

When this reference is used, execute the following steps in order.

1. **Use package check, computational run, dataset analysis, parameter sweep, validation result, and claim records**.
2. **Link claims to supporting evidence rather than relying on prose**.

If the user's task does not map cleanly to these steps, use your native planning tool to build a step-by-step plan from this reference, the parent skill, and the available evidence, then execute the plan.
