# Claim Discipline

Use this reference to calibrate scientific claims to evidence type. Placeholder definitions live in `../migrate/placeholders.md`.

## Workflow

When this reference is used, execute the following steps in order.

1. **Computed claims need current-run evidence**.
2. **Parsed claims need source and parser evidence**.
3. **Digitized claims need extraction provenance**.
4. **Hypotheses must remain hypotheses until upgraded by evidence**.

If the user's task does not map cleanly to these steps, use your native planning tool to build a step-by-step plan from this reference, the parent skill, and the available evidence, then execute the plan.
