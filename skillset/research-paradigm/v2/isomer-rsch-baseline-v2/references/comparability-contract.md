# Comparability Contract

Use this reference to define the comparison basis that later work must not guess. Placeholder definitions live in `../migrate/placeholders.md`.

## Workflow

When this reference is used, execute the following steps in order.

1. **Record task, dataset, split, metric ids, directions, source identity, evaluation path, expected outputs, deviations, and caveats**.
2. **Classify verdicts as verified match, verified close, verified diverged, trusted with caveats, broken, waived, or blocked**.

If the user's task does not map cleanly to these steps, use your native planning tool to build a step-by-step plan from this reference, the parent skill, and the available evidence, then execute the plan.
