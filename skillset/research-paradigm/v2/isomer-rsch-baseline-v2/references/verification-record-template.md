# Verification Record Template

Use this reference to capture baseline evidence before acceptance. Placeholder definitions live in `../migrate/placeholders.md`.

## Workflow

When this reference is used, execute the following steps in order.

1. **State route, comparator identity, command or source evidence, output pointers, metric values, environment facts, deviations, verdict, and caveats**.
2. **Do not treat attach, import, or publish as acceptance without verification**.

If the user's task does not map cleanly to these steps, use your native planning tool to build a step-by-step plan from this reference, the parent skill, and the available evidence, then execute the plan.
