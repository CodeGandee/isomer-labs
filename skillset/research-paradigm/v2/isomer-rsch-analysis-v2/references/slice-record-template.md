# Analysis Slice Record Template

Use this reference to record one follow-up analysis slice before campaign-level interpretation. Placeholder definitions live in `../migrate/placeholders.md`.

## Workflow

When this reference is used, execute the following steps in order.

1. **Fill parent object, slice question, intervention or inspection target, metric or observable, evidence source, comparability verdict, claim update, caveat, and next action**.
2. **Classify the result as supports, weakens, refutes, clarifies, or blocks the parent claim**.
3. **Do not merge multiple interventions into one slice record**.

If the user's task does not map cleanly to these steps, use your native planning tool to build a step-by-step plan from this reference, the parent skill, and the available evidence, then execute the plan.
