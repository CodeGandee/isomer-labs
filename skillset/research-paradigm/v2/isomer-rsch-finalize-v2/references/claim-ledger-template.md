# Claim Ledger Template

Use this reference to classify final claim status. Placeholder definitions live in `../migrate/placeholders.md`.

## Workflow

When this reference is used, execute the following steps in order.

1. **For each claim, record evidence, caveats, safe-to-surface status, downgrade history, and recommendation**.
2. **Preserve weakened or failed claims rather than deleting them**.

If the user's task does not map cleanly to these steps, use your native planning tool to build a step-by-step plan from this reference, the parent skill, and the available evidence, then execute the plan.
