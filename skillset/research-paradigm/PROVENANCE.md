# Provenance

The research-paradigm skills are derived from local method analysis notes that summarized DeepScientist source skill behavior.

The retired first-generation adaptation preserved the first Isomer Labs pass over those DeepScientist-derived skills. It kept richer stage guidance, paper-facing companions, storage vocabulary, lifecycle vocabulary, provenance context, and license notices with generation-suffixed skill names.

The production DeepSci generation under `deepsci/` distills the core research-conduction process from `context/explore/deepscientist-skill-analysis/`. It keeps the research loop and guardrails, then replaces storage-facing outputs with semantic placeholders so their meaning can be settled before storage binding.

The production paper-writing batch covers the Paper Writing Skills in that analysis index: write, paper-outline, paper-plot, figure-polish, review, rebuttal, nature-data, nature-figure, nature-paper2ppt, and nature-polishing. Each migrated skill keeps the upstream source copy under `org/src/`, the copied process analysis under `org/analysis/`, and local refactor-migration notes under `migrate/`.

The active runtime surface for a production DeepSci skill is its `SKILL.md`, manifest, directly linked active references, active assets, and active scripts. `migrate/`, `org/analysis/`, `org/src/`, passive templates, provenance notes, license notices, and deferred-resource notes are traceability material for reviewers and maintainers, not active installed-skill guidance.

The local DeepScientist source checkout and earlier inspection notes are provenance for maintainers, not runtime dependencies for installed skills.
