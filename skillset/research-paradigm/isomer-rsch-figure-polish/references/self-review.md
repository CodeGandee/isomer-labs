# Self-Review

Use a mandatory render-inspect-revise loop for any figure intended for milestone reporting, manuscript drafting, appendix use, review response, or durable Artifact storage.

## Required Loop

1. Render a first draft.
2. Open or inspect the rendered file with an available image or file inspection capability.
3. Review the actual image, not just the plotting code.
4. Revise the figure-generation Artifact if readability, composition, or evidence linkage is weak.
5. Re-export the final version and record what changed.

## Checklist

- Is the main message obvious in a few seconds?
- Are labels, units, baselines, and uncertainty semantics explicit?
- Is the legend unnecessary, too large, or blocking data?
- Does text remain readable after realistic down-scaling?
- Is the main method visually dominant only when the evidence warrants it?
- Are line widths, marker sizes, error bars, and bands balanced?
- Are ticks too dense or labels colliding?
- Is the grid helpful rather than decorative?
- Would the figure remain understandable in grayscale or for color-vision-deficient readers?
- Does the figure avoid decorative clutter that does not help the claim?

If any answer is negative, revise before calling the figure complete.
