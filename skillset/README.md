# Project Skills

Project-specific agent skills live in this directory.

Name skills as `isomer-labs-<what>`, where `<what>` describes the operation or
capability, such as `isomer-labs-review` or `isomer-labs-release`.

Research-paradigm skills live under `skillset/research-paradigm/`. They use the
extended name form `isomer-labs-research-<purpose>` so related research-stage
skills stay grouped while preserving the project prefix.
