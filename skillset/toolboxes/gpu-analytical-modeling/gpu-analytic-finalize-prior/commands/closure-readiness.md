# Closure Readiness

Before finalization, identify central contribution claims and check whether matching evidence is visible. Runtime, counter-trend, saturated-component, and blocking-path claims each need matching evidence or an explicit limitation.

Choose closure posture by the weakest central claim: publish, park, defer, limit, route back, or block. If parked or deferred, name the evidence, implementation, or access condition that would change the decision.

Preserve resume context: what to read first, which failed or partial evidence must not be repeated blindly, and which proxy evidence must stay labeled as proxy support.
