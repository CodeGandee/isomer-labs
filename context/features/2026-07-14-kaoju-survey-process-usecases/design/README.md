# Design

The authoritative proposed architecture, interface contracts, alternatives, risks, and migration plan live in the [OpenSpec design](../../../../openspec/changes/revise-kaoju-survey-process/design.md). The accepted ADRs in the adjacent `adrs/` directory remain the decision constraints for that design.

This directory does not maintain a second module map or interface contract because parallel design authorities would drift. Feature-level refinements should update the OpenSpec change and the affected use case or ADR together.
