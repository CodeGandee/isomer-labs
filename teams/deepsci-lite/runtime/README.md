# DeepScientist Lite Runtime Runbooks

This directory holds launch scripts, operator runbooks, and stage-transition checklists for the `deepsci-lite` team.

## Contents

- `launch-foundation.md`: launch `deepsci-foundation` and review the scout/baseline output.
- `launch-strategist.md`: hand off to `deepsci-strategist` after baseline acceptance.
- `launch-executor.md`: hand off to `deepsci-executor` after idea selection.
- `launch-synthesis.md`: hand off to `deepsci-synthesis` after experiment validation.
- `call-lead.md`: ask `deepsci-lead` for a transition decision at any major gate.

Add runbooks as the team matures.
