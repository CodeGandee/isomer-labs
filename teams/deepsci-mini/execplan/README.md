# deepsci-mini Execplan

## Purpose

Generated Houmao loop-pro operational material for the `deepsci-mini` Domain Agent Team Template. Editable source lives under `../intention/`; this directory is derived material.

## Contents

- `manifest.toml`: generated package metadata, source posture, stage status, and artifact index.
- `specs/`: objective, collaboration, topology, communication, state, workspace, run, and participant contracts.
- `skills/`: generated shared, event, tick, and operator-control skills.
- `agents/`: generated Houmao profile material, bindings, and notifier prompts.
- `harness/`: generated validation, query, mail schema, control, and placeholder helper surface.
- `docs/`: generated human support views.
