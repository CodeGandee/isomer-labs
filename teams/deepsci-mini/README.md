# deepsci-mini

`deepsci-mini` is a compact Domain Agent Team Template for UC-01, Explore a New Research Direction. It keeps the DeepScientist-style research loop small: one lead, one scout, and one synthesis reviewer.

## Contents

- `source/`: design source for the mini team.
- `intention/`: editable Houmao loop-pro intention source.
- `execplan/`: generated Houmao operational material.
- `runs/`: optional local generated-loop run state if a later execution stage prepares it.

## Boundary

This package is not a Topic Workspace and not Workspace Runtime state. A concrete Research Topic must specialize this template into a Topic Agent Team Profile before Isomer creates an Agent Team Instance.
